## 使用说明

所有Logo文件均为**SVG格式**，具有以下特点：

- **矢量格式**：可无限缩放，不失真
- **轻量级**：文件大小小，加载快速
- **可编辑**：可用文本编辑器或设计软件修改
- **兼容性**：支持所有现代浏览器和设计工具

### 集成方式

#### HTML中使用
```html
<img src="aliexpress.svg" alt="AliExpress" width="100" height="100">
```

#### CSS背景图
```css
.logo {
  background-image: url('shopify.svg');
  background-size: contain;
  background-repeat: no-repeat;
}
```

#### React/Vue组件
```jsx
import AliExpressLogo from './aliexpress.svg';

function App() {
  return <img src={AliExpressLogo} alt="AliExpress" />;
}
```
